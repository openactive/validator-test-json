#### Example Rule 9 and Rule 5

Not like this:

```javascript
"offer": [
  {
    "type": "Offer",
    "name": "Full price cost",
    "price": 45,
    "priceCurrency": "GBP",
    "url": ""
  }
]
```

Like this:

```javascript
"offer": [
  {
    "type": "Offer",
    "name": "Full price cost",
    "price": "45.00", /* Price should always be a string with 2 decimal places */
    "priceCurrency": "GBP"
  }
]
```
